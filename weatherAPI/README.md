# Weather App (Fetch API Project)

A simple and modern **Weather App** built using **HTML, CSS, and JavaScript** that fetches live weather data from the **OpenWeatherMap API** using the **Fetch API**. This project demonstrates how real-world web applications communicate with servers and update the user interface dynamically without reloading the page.

---

## Features

- Search weather by city name  
- Live temperature in Celsius  
- Weather description (cloudy, sunny, rainy, etc.)  
- Feels-like temperature  
- Weather icons from OpenWeatherMap  
- Loading and error handling  
- Clean and responsive UI  

## Concepts Used

- Fetch API  
- Promises  
- Asynchronous JavaScript  
- API Requests  
- DOM Manipulation  
- Event Handling  

## API Used

This project uses the **OpenWeatherMap API** to get live weather data.

Website:  
https://openweathermap.org/
